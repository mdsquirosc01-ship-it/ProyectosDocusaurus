# Programación de Multimedia y Dispositivos Móviles (PMDM)

¡Bienvenido al módulo de **Programación de Multimedia y Dispositivos Móviles**! En esta asignatura, exploraremos el fascinante mundo del desarrollo de aplicaciones para dispositivos Android, aprendiendo desde los conceptos básicos hasta la creación de apps profesionales y robustas.

## 📱 ¿Qué aprenderemos?

A lo largo del curso, nos centraremos en los pilares fundamentales del desarrollo móvil moderno:

- **Entorno de desarrollo:** Configuración y dominio de **Android Studio**.
- **Lenguaje de programación:** Utilizaremos **Kotlin**, el lenguaje oficial y preferido para el desarrollo en Android.
- **Interfaces de Usuario (UI):** Diseño de interfaces atractivas y funcionales utilizando XML y, opcionalmente, Jetpack Compose.
- **Ciclo de vida:** Entender cómo funcionan las aplicaciones en segundo plano y la gestión de recursos.
- **Persistencia de datos:** Cómo guardar información localmente usando SQLite o Room.
- **Conectividad:** Consumo de APIs y servicios web para crear aplicaciones conectadas.

## 🛠️ Herramientas Necesarias

Para seguir este módulo de forma óptima, necesitaremos:

1. **Android Studio:** El IDE oficial.
2. **JDK (Java Development Kit):** Generalmente incluido en Android Studio.
3. **Dispositivo físico o Emulador:** Para probar nuestras creaciones.

:::tip Recomendación
Es muy recomendable disponer de un dispositivo físico Android para realizar las pruebas, ya que el rendimiento es mucho más fluido que en el emulador.
:::
